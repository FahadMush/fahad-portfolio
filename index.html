Logistics Data Transformation and Analysis

This project, "Logistics Data Transformation and Analysis," addresses the challenge of optimizing transportation costs across various modes (air, road, sea, rail) by integrating fragmented data from disparate sources. The core problem involved inconsistent data formats and naming conventions across cost tables, and the need to accurately calculate per-lane transportation costs while preserving all original operational details.

Through advanced SQL techniques, including CREATE TABLE AS SELECT for data transformation, UNION ALL for unpivoting, and conditional LEFT JOIN operations with COALESCE() for intelligent cost aggregation, a unified TotalTransportationCost dataset was created.

This consolidated dataset provides comprehensive, lane-specific transportation costs, enabling stakeholders to identify high-cost lanes, compare mode efficiencies, and make data-driven decisions for cost optimization and enhanced supply chain efficiency.
